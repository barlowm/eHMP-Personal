Navigation

Description: The navigation is displayed at the top of the application allowing the user to perform actions such as searching for a patient or signing out of the application.

508 Guidelines: The navigation contains links; therefore, to pass 508, all links must be keyboard accessible, and the screen reader should read each link. Users can navigate to the links in the drop down menu by tabbing to the menu and then pressing the up and down arrow keys.

TIP: To place the navigation at the top and make it fixed, add the class "navbar-fixed-top" to the nav tag
